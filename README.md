# Madevisibly Marketing Site

A clean, modern, conversion-focused landing page for Madevisibly — a marketing agency for local service businesses.

## What it includes
- Hero section with strong lead-gen CTA
- Services section for Google presence, websites, social, and ads
- Pricing section with 3 tiers
- Mobile-friendly audit/contact form
- Scroll reveal, progress bar, marquee, and subtle motion

## Pricing CTA
Primary CTA across the site:
- **Book a Free Audit**

## Run locally
Open `index.html` in a browser or serve the folder with any static file server.

## Notes
This page is intentionally simple so it can be dropped into most static hosting setups or adapted into Next.js, Astro, or Vite later.
