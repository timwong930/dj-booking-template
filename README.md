# DJ Booking Template

A dark, premium, motion-light website template for DJs who book parties, clubs, bars, weddings, and private events.

## What's inside
- Hero section with a strong booking CTA
- Services section for DJ event types
- Social link hub for Instagram, TikTok, SoundCloud, YouTube, etc.
- Contact info section for phone and email
- Inquiry form section for event booking leads
- Subtle scroll reveal motion

## Replace these placeholders
- `Nightwave` brand name
- Social URLs and handles
- Phone number
- Email address
- Formspree endpoint in the contact form
- Any photos, mixes, or embeds you want to add

## Suggested CTA copy
Use one primary CTA across the site:
- **Inquire for your event**
- **Check DJ availability**
- **Book your event DJ**

## Run locally
Open `index.html` in a browser or serve the folder with any static file server.

## Notes
This template is intentionally simple so it can be dropped into most static hosting setups or adapted into Next.js, Astro, or Vite later.
